# 📌 Social Network Data Analysis & Recommendation System

A pure Python-based project for analyzing social network data and building a recommendation system.

## 🚀 Overview
This project analyzes user interactions and relationships to:
- Suggest friends ("People You May Know")
- Recommend pages based on interests
- Understand user behavior

## 📂 Project Files
- intro.ipynb – Introduction and dataset overview  
- data_cleaning.ipynb – Data preprocessing  
- people_you_may_know.ipynb – Friend recommendation  
- pages_you_like.ipynb – Page recommendation  

## 🧠 Features
- Social network graph analysis  
- Friend recommendation using mutual connections  
- Page recommendation system  
- Implemented using core Python  

## ⚙️ Technologies
- Python  
- Jupyter Notebook  

## ▶️ How to Run
1. Clone the repo:
   git clone https://github.com/your-username/social-network-analysis.git

2. Open Jupyter Notebook:
   jupyter notebook

3. Run all notebooks step by step.
